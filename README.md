# Bank Course Work 1

Домашнее задание по анализу банковских данных.

## Установка

1. Установите Poetry: https://python-poetry.org/docs/#installation
2. Установите зависимости:
   ```bash
   poetry install```
3.Запуск
```poetry run python main.py```
4.Тестирование
Запуск тестов:
```poetry run pytest```
5.Проверка стиля кода:
```poetry run flake8```
